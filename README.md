🔍 Inverted Search in C

A console-based Inverted Search application developed in C that efficiently indexes and searches words across multiple text files using Hashing and Linked Lists. This project was built as part of my Embedded Systems learning journey and focuses on data structures, file handling, hashing techniques, and efficient searching in C.

🚀 Project Overview

An Inverted Search is a data structure widely used in search engines to quickly locate information. Instead of searching every file each time, this project creates an indexed database that stores each unique word along with the files in which it appears and the number of occurrences.

The application allows users to:

Create an indexed database from multiple text files
Search words efficiently
Display the complete database
Save the database to a backup file
Restore the database from a saved backup
